My setup scripts
=============

Setup scripts for development environment on Linux(CentOS or Ubuntu).

## Setup

The recommended setup order is as follows:

```
./setup_ghq.sh
./setup_dotfiles.sh
./setup_ruby.sh
./setup_git.sh
./setup_zsh.sh
./setup_vim.sh
./setup_tmux.sh
```
